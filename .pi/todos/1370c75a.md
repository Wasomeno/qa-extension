#### Usage:
- **Modal mode**: Left-click the menu icon (default behavior)
- **Page mode**: Middle-click the menu icon (opens in new tab)

#### Architecture:
- `MainMenuContent` component supports both `modal` and `page` display modes
- User session is shared automatically via `chrome.storage.session`
- Navigation state is independent per instance (can be enhanced with sync if needed)
- Page mode includes a top bar with logo, refresh, and close buttons

#### Build:
- Run `npm run build:chrome` or `npm run build:firefox`
- main-menu.html and main-menu.js are generated in dist/
